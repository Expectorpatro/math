\chapter{机器学习}

\gls{ML}旨在使模型从数据中学习潜在的规律。根据训练数据是否包含需要预测的目标信息，机器学习方法可以大致分为\gls{SupervisedLearning}和\gls{UnsupervisedLearning}。\par
监督学习的训练数据由输入变量及其对应的输出结果组成。模型通过学习输入与输出之间的关系，对新的输入进行预测。例如，已知若干房屋的面积、位置及其价格，可以学习房屋特征与房价之间的关系，并预测新房屋的价格。\par
无监督学习的训练数据只包含输入变量，而不包含预先给定的输出结果。其目标不是预测某个已知标签，而是从数据中发现潜在的结构或模式。例如，可以根据样本之间的相似性将其划分为若干组，或者用较少的变量概括原始数据的主要信息。\par
在监督学习中，根据输出变量的类型，可以进一步将问题分为分类问题和回归问题。\par
分类问题的输出是离散的类别。模型需要根据输入变量判断一个样本属于哪个类别。例如，根据邮件内容判断其是否为垃圾邮件，或者根据医学检查结果判断患者是否患病。\par
回归问题的输出是连续的数值。模型需要学习输入变量与数值型输出之间的关系。例如，根据房屋面积和位置预测房价，或者根据患者的临床指标预测某项连续的生理数值。
